theAudioDB

bereitgestellt durch RapidAPI

Referenz für API

Beispielcode:
const options = {
method: 'GET',
mode: 'cors',
headers: {
'X-RapidAPI-Key': '570e25dff9mshc40a93d4eac80d3p1add41jsn37a8fcb00570',
'X-RapidAPI-Host': 'theaudiodb.p.rapidapi.com'
}
};

fetch('https://theaudiodb.p.rapidapi.com/track.php?h=32793500', options)
.then(response => response.json())
.then(response => console.log(response))
.catch(err => console.error(err));

Planned Use-Cases and Endpoints:
  • Gather ID of a Song that is a cover (Endpoint: GET "Return track details from artist/track name")
  • Request Infos about a Song that is a cover with ID (Endpoint: GET "Return individual track info using a known TADBTrackID")

Shazam Core


bereitgestellt durch RapidAPI

Referenz für API
Beispielcode:
const data = new Buffer(); const options = {
method: 'POST',
mode: 'cors',
headers: {
'content-type': 'application/json',
'X-RapidAPI-Key': '570e25dff9mshc40a93d4eac80d3p1add41jsn37a8fcb00570',
'X-RapidAPI-Host': 'shazam-core.p.rapidapi.com'
},
body: data
};

fetch('https://shazam-core.p.rapidapi.com/v1/tracks/recognize', options)
.then(response => response.json())
.then(response => console.log(response))
.catch(err => console.error(err));

Planned Use-Cases and Endpoints:
  • Recognise a Track to tell if it is safe to post the Song on Youtube, SoundCloud etc. without getting flagged for Copyright violation (Endpoint: POST "Track Recognition")

Parameter:

  • body - binary data of a audio file in MP3,WAV,etc... (3-5 seconds, no more than 1 mb, best 500kb)
